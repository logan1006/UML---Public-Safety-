Welcome!

This DashBoard deals with the following five Datasets in the city of Chicago:

       Crime Data
       Life Safety Evaluation Data
       Police Station Data
       Fire Station Data
       Health Data

-You can view the data of these Datasets by selecting any of these Datasets from the list beside.

-While viewing the data, you can plot the Charts of five types viz., Bar, Line, Doughnut, Polar and Pie, you can Export the data in CSV format. You can also filter the data by specific conditions.

-You can navigate to any page of the data set from any page.
